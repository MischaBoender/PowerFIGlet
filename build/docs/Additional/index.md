## Installation
PowerFIGlet is available on the [PowerShell Gallery](https://www.powershellgallery.com/packages/PowerFIGlet/).

To Inspect:
```powershell
Save-Module -Name PowerFIGlet -Path <path>
```
To install:
```powershell
Install-Module -Name PowerFIGlet -Scope CurrentUser
```
